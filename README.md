# ChatInterface

A React Native chat interface application built with Expo.

## Features
- Real-time chat interface
- QR code pairing
- Voice messaging support
- Firebase integration
- Cross-platform (iOS/Android)

## Tech Stack
- React Native
- Expo
- TypeScript
- Firebase
- Socket.io

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- Expo CLI
- Android Studio (for Android development)
- Xcode (for iOS development)

### Installation

1. Clone the repository
```bash
git clone https://github.com/divyanshthakur5552/chatinterface.git
cd chatinterface
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```
